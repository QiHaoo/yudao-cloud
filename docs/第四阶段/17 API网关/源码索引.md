## 业务层使用示例

### 路由配置

```yaml
spring:
  cloud:
    gateway:
      routes:
        - id: system-admin-api
          uri: grayLb://system-server
          predicates:
            - Path=/admin-api/system/**
          filters:
            - RewritePath=/admin-api/system/v3/api-docs, /v3/api-docs
        - id: product-admin-api
          uri: grayLb://product-server
          predicates:
            - Path=/admin-api/product/**
```

### Token 校验流程

```
请求: GET /admin-api/system/user/get?id=1
Header: Authorization: Bearer xxx

TokenAuthenticationFilter:
  1. 提取 Token
  2. 从缓存获取 LoginUser（缓存 Key: tenantId + token）
  3. 缓存未命中时调用 system-server 校验
  4. 注入 login-user Header
  5. 转发到 system-server
```

### 灰度路由

```
请求 Header: version=2.0.0

GrayLoadBalancer:
  1. 获取 system-server 所有实例
  2. 过滤 metadata.version == 2.0.0 的实例
  3. 随机+权重选择一个实例
```

## 配置项速查

### 路由配置

| 配置项 | 作用 | 示例值 |
|--------|------|--------|
| `spring.cloud.gateway.routes[].id` | 路由 ID | `system-admin-api` |
| `spring.cloud.gateway.routes[].uri` | 目标服务 | `grayLb://system-server` |
| `spring.cloud.gateway.routes[].predicates` | 匹配条件 | `Path=/admin-api/system/**` |
| `spring.cloud.gateway.routes[].filters` | 过滤器 | `RewritePath=...` |
| `spring.cloud.gateway.default-filters` | 全局过滤器 | `DedupeResponseHeader=...` |
| `spring.cloud.gateway.x-forwarded.prefix-enabled` | 前缀转发 | `false` |

### 端口配置

| 配置项 | 作用 | 示例值 |
|--------|------|--------|
| `server.port` | 网关端口 | `48080` |
| `spring.codec.max-in-memory-size` | 缓冲区大小 | `10MB` |

### Knife4j 配置

```yaml
knife4j:
  gateway:
    enabled: true
    routes:
      - name: system-server
        service-name: system-server
        url: /admin-api/system/v3/api-docs
```

### URL 前缀规范

| 类型 | 前缀 | 路由到 |
|------|------|--------|
| 管理后台 | `/admin-api/{module}/...` | `{module}-server` |
| 用户端 | `/app-api/{module}/...` | `{module}-server` |
| RPC 调用 | `/rpc-api/{module}/...` | 不经过网关 |

### 过滤器执行顺序

| 过滤器 | Order | 说明 |
|--------|-------|------|
| TokenAuthenticationFilter | -100 | Token 校验 |
| CorsFilter | WebFilter（自动） | 跨域处理 |
| AccessLogFilter | 0 | 访问日志 |
| GrayReactiveLoadBalancerClientFilter | 10150 | 灰度负载均衡 |

## 业务层使用示例

### 服务注册配置

```yaml
# application-local.yaml
spring:
  cloud:
    nacos:
      server-addr: 127.0.0.1:8848
      username: nacos
      password: nacos-admin
      discovery:
        namespace: dev
        group: DEFAULT_GROUP
        metadata:
          version: 1.0.0  # 用于灰度发布
```

### 配置中心配置

```yaml
# application-local.yaml
spring:
  cloud:
    nacos:
      config:
        namespace: dev
        group: DEFAULT_GROUP
        shared-configs:
          - data-id: application-common.yaml
            group: SHARED_GROUP
            refresh: true
  config:
    import:
      - optional:classpath:application-${spring.profiles.active}.yaml
      - optional:nacos:${spring.application.name}-${spring.profiles.active}.yaml
```

### 单体模式禁用配置

```yaml
# yudao-server application.yaml
spring:
  cloud:
    nacos:
      discovery:
        enabled: false
      config:
        enabled: false
```

### 灰度发布配置

```yaml
# 新版本实例
spring:
  cloud:
    nacos:
      discovery:
        metadata:
          version: 2.0.0
```

## 配置项速查

### 服务注册配置

| 配置项 | 作用 | 示例值 |
|--------|------|--------|
| `spring.cloud.nacos.server-addr` | Nacos 服务器地址 | `127.0.0.1:8848` |
| `spring.cloud.nacos.username` | 账号 | `nacos` |
| `spring.cloud.nacos.password` | 密码 | `nacos-admin` |
| `spring.cloud.nacos.discovery.namespace` | 命名空间 | `dev` |
| `spring.cloud.nacos.discovery.group` | 分组 | `DEFAULT_GROUP` |
| `spring.cloud.nacos.discovery.metadata` | 实例元数据 | `version: 1.0.0` |
| `spring.cloud.nacos.discovery.enabled` | 是否启用 | `true` / `false` |

### 配置中心配置

| 配置项 | 作用 | 示例值 |
|--------|------|--------|
| `spring.cloud.nacos.config.namespace` | 命名空间 | `dev` |
| `spring.cloud.nacos.config.group` | 分组 | `DEFAULT_GROUP` |
| `spring.cloud.nacos.config.shared-configs` | 共享配置列表 | 见下方 |
| `spring.cloud.nacos.config.enabled` | 是否启用 | `true` / `false` |
| `spring.config.import` | 配置导入 | `optional:nacos:xxx.yaml` |

### 共享配置格式

```yaml
spring:
  cloud:
    nacos:
      config:
        shared-configs:
          - data-id: application-common.yaml
            group: SHARED_GROUP
            refresh: true
```

### 配置加载优先级

```
1. Nacos: ${service-name}-${profile}.yaml    ← 最高
2. Nacos: shared-configs
3. 本地: application-${profile}.yaml
4. 本地: application.yaml
5. 代码: @Value 默认值                        ← 最低
```

## 常见问题排查速查

| 问题现象 | 可能原因 | 排查步骤 |
|----------|----------|----------|
| 服务注册失败 | Nacos 地址错误 | 检查 server-addr，telnet 测试连通性 |
| 配置未生效 | Data ID 不匹配 | 检查 `${spring.application.name}-${profile}.yaml` |
| 配置未热更新 | 未使用 @RefreshScope | 在 Bean 上添加注解 |
| 启动报错连接失败 | Nacos 不可用且未使用 optional | 使用 `optional:nacos:` 前缀 |
| Namespace 不匹配 | 控制台配置与代码不一致 | 检查 namespace ID 是否正确 |
| 共享配置未加载 | Group 不匹配 | 检查 shared-configs 中的 group |

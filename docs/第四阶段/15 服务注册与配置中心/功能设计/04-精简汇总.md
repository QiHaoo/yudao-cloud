# 04 - 精简汇总

## 配置项速查

### 服务注册配置

```yaml
spring:
  cloud:
    nacos:
      server-addr: 127.0.0.1:8848    # Nacos 服务器地址
      username: nacos                 # 账号
      password: nacos-admin           # 密码
      discovery:
        namespace: dev                # 命名空间（环境隔离）
        group: DEFAULT_GROUP          # 分组
        metadata:
          version: 1.0.0              # 服务实例版本（灰度发布）
```

### 配置中心配置

```yaml
spring:
  cloud:
    nacos:
      config:
        namespace: dev                # 命名空间
        group: DEFAULT_GROUP          # 分组
        shared-configs:               # 共享配置
          - data-id: application-common.yaml
            group: SHARED_GROUP
            refresh: true
  config:
    import:
      - optional:classpath:application-${spring.profiles.active}.yaml
      - optional:nacos:${spring.application.name}-${spring.profiles.active}.yaml
```

### 单体模式禁用配置

```yaml
spring:
  cloud:
    nacos:
      discovery:
        enabled: false
      config:
        enabled: false
```

## 核心设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 注册中心选型 | Nacos | 同时支持服务注册和配置中心，Spring Cloud Alibaba 原生集成 |
| 配置加载方式 | spring.config.import | Spring Boot 2.4+ 推荐，支持 optional 降级 |
| 环境隔离 | namespace | 物理隔离，不同环境数据完全独立 |
| 共享配置 | shared-configs | 减少重复配置，统一管理公共配置 |
| 一致性模型 | AP（服务发现）+ CP（配置管理） | 服务发现重可用，配置管理重一致 |
| 部署模式 | 单体/微服务可切换 | 同一套代码，通过配置切换部署模式 |

## Nacos 控制台操作速查

| 操作 | 路径 | 说明 |
|------|------|------|
| 查看服务列表 | 服务管理 → 服务列表 | 查看所有注册的服务 |
| 查看实例详情 | 服务详情 → 实例列表 | 查看服务的实例和元数据 |
| 查看配置列表 | 配置管理 → 配置列表 | 查看所有配置文件 |
| 创建命名空间 | 命名空间 → 新建 | 创建新的环境隔离空间 |
| 创建配置 | 配置管理 → 新建 | 创建新的配置文件 |
| 配置监听 | 配置管理 → 监听查询 | 查看哪些实例在监听配置 |

## 配置加载优先级

```
优先级从高到低：
  1. Nacos 上的 ${service-name}-${profile}.yaml
  2. Nacos 上的 shared-configs
  3. 本地 application-${profile}.yaml
  4. 本地 application.yaml
  5. 代码中的 @Value 默认值
```

## 常见问题排查

| 问题 | 可能原因 | 排查方法 |
|------|----------|----------|
| 服务注册失败 | Nacos 地址错误、网络不通 | 检查 server-addr 配置，telnet 测试连通性 |
| 配置未生效 | namespace 不匹配、Data ID 不正确 | 检查 Nacos 控制台的配置列表 |
| 配置未热更新 | 未使用 @RefreshScope | 在 Bean 上添加 @RefreshScope 注解 |
| 启动报错 | Nacos 不可用但未使用 optional | 使用 `optional:nacos:` 前缀 |

# 03 - 测试支持与源码索引

## 一、测试支持

### 1.1 测试基础设施

工作流模块的测试依赖以下基础设施：

1. **H2 内存数据库**：单元测试使用 H2 内存数据库，同时需要初始化 Flowable 的表结构。
2. **Flowable 测试支持**：Flowable 提供了 `flowable-spring-boot-starter-test` 依赖，支持流程测试。
3. **Spring Boot Test**：使用 `@SpringBootTest` 注解加载完整的 Spring 上下文。

### 1.2 现有测试

| 测试类 | 覆盖范围 |
|--------|---------|
| `BpmCategoryServiceImplTest` | 流程分类的 CRUD |
| `BpmFormServiceTest` | 表单的 CRUD |

### 1.3 常见测试场景

#### 场景一：流程分类 CRUD 测试

```java
@SpringBootTest
class BpmCategoryServiceImplTest {

    @Resource
    private BpmCategoryService categoryService;

    @Test
    public void testCreateCategory() {
        BpmCategoryCreateReqVO reqVO = new BpmCategoryCreateReqVO()
                .setName("OA 审批")
                .setCode("oa")
                .setSort(1);
        Long id = categoryService.createCategory(reqVO);

        BpmCategoryDO category = categoryService.getCategory(id);
        assertNotNull(category);
        assertEquals("OA 审批", category.getName());
    }

    @Test
    public void testUpdateCategory() {
        Long id = createTestCategory();
        BpmCategoryUpdateReqVO reqVO = new BpmCategoryUpdateReqVO()
                .setId(id)
                .setName("财务审批");
        categoryService.updateCategory(reqVO);

        BpmCategoryDO category = categoryService.getCategory(id);
        assertEquals("财务审批", category.getName());
    }
}
```

#### 场景二：表单 CRUD 测试

class BpmFormServiceTest {

### 1.5 测试数据准备

```java
// 测试基类
@SpringBootTest
public abstract class BpmBaseTest {

    @Resource
    protected RepositoryService repositoryService;
    @Resource
    protected RuntimeService runtimeService;
    @Resource
    protected TaskService taskService;
    @Resource
    protected HistoryService historyService;

    // 创建测试流程定义
    protected ProcessDefinition createTestProcessDefinition(String key) {
        String bpmnXml = "<?xml version=\"1.0\" encoding=\"UTF-8\"?>\n" +
                "<definitions xmlns=\"http://www.omg.org/spec/BPMN/20100524/MODEL\">\n" +
                "  <process id=\"" + key + "\" name=\"测试流程\" isExecutable=\"true\">\n" +
                "    <startEvent id=\"start\"/>\n" +
                "    <userTask id=\"approve\" name=\"审批\" flowable:assignee=\"${assignee}\"/>\n" +
                "    <endEvent id=\"end\"/>\n" +
                "    <sequenceFlow sourceRef=\"start\" targetRef=\"approve\"/>\n" +
                "    <sequenceFlow sourceRef=\"approve\" targetRef=\"end\"/>\n" +
                "  </process>\n" +
                "</definitions>";

        Deployment deployment = repositoryService.createDeployment()
                .addString(key + ".bpmn20.xml", bpmnXml)
                .deploy();

        return repositoryService.createProcessDefinitionQuery()
                .deploymentId(deployment.getId())
                .singleResult();
    }

    // 创建测试流程实例
    protected ProcessInstance createTestProcessInstance(String processDefinitionId, Long assignee) {
        return runtimeService.createProcessInstanceBuilder()
                .processDefinitionId(processDefinitionId)
                .variable("assignee", String.valueOf(assignee))
                .start();
    }
}
```

# 03 - 精简汇总

## 报表流程速查

### JimuReport 报表设计流程

```
配置数据源 → 创建数据集(SQL/API/Java) → 设计模板 → 预览 → 发布
```

### GoView 大屏设计流程

```
创建项目 → 拖拽组件 → 绑定数据接口 → 调整样式 → 发布
```

### 报表导出流程

```
用户点击导出 → 执行数据查询 → 渲染模板 → 生成文件 → 下载
```

---

## 两种报表工具对比

| 维度 | JimuReport | GoView |
|------|-----------|--------|
| 定位 | 传统报表 | 数据大屏 |
| 场景 | 财务报表、运营报表 | 监控大屏、展示大屏 |
| 格式 | 表格为主 | 图表为主 |
| 交互 | 查询、导出、打印 | 查看、刷新 |
| 使用者 | 运营、财务 | 管理层、展厅 |
| 导出 | Excel/PDF/Word | 截图 |

## 核心 API 速查

### JimuReport（积木报表）

|------|------|------|
| `/jmreport/list` | GET | 报表列表 |

| `/report/go-view/project/get` | GET | 查询项目详情 |

## 核心设计决策

| 决策 | 选择 | 放弃的方案 | 原因 |
|------|------|-----------|------|
| 报表引擎 | JimuReport | 自研报表 | 开源成熟，功能完善 |
| 数据大屏 | GoView | 自研大屏 | 开源免费，可视化丰富 |
| 集成方式 | 内嵌模式 | 独立部署 | 简化部署，共享数据源 |
| 数据集 | SQL + API + Java | 只支持 SQL | 灵活适配不同场景 |
| 导出格式 | Excel + PDF + Word | 只支持 Excel | 满足不同使用需求 |
| 数据刷新 | 定时轮询 | WebSocket 推送 | 实现简单，兼容性好 |

---

## 数据集类型

| 类型 | 适用场景 | 示例 |
|------|---------|------|
| SQL 数据集 | 直接查询业务表 | `SELECT * FROM orders` |
| API 数据集 | 调用微服务接口 | `GET /api/sales/trend` |
| Java 数据集 | 复杂计算逻辑 | `SalesService.calc()` |

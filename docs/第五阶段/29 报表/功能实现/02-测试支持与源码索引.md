# 02-测试支持与源码索引

## 测试支持

### 1. 测试策略概述

报表模块的测试面临以下挑战：

1. **外部依赖**：JimuReport 是第三方库，无法直接 Mock 其内部实现
2. **Token 适配**：需要测试 Token 验证逻辑与 OAuth2 体系的集成
3. **数据库依赖**：报表数据集查询依赖真实的数据库连接

采用的测试策略：
1. **Mock OAuth2TokenCommonApi**：模拟 Token 验证
2. **Mock PermissionCommonApi**：模拟权限查询
3. **集成测试**：验证完整的报表访问流程

    @Test
    void testVerifyTokenWithExistingUser() {
        // Given
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUserId).thenReturn(1L);

        // When
        Boolean result = tokenService.verifyToken("test-token");

        // Then
        assertTrue(result);
        verifyNoInteractions(oauth2TokenApi);
    }

    @Test
    void testVerifyTokenWithValidToken() {
        // Given
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUserId).thenReturn(null);
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUser).thenReturn(null);

        OAuth2AccessTokenCheckRespDTO tokenResp = new OAuth2AccessTokenCheckRespDTO();
        tokenResp.setUserId(1L);
        tokenResp.setUserType(2);
        tokenResp.setTenantId(1L);

        when(oauth2TokenApi.checkAccessToken("valid-token"))
                .thenReturn(CommonResult.success(tokenResp));

        // When
        Boolean result = tokenService.verifyToken("valid-token");

        // Then
        assertTrue(result);
        verify(oauth2TokenApi).checkAccessToken("valid-token");
    }

    @Test
    void testVerifyTokenWithInvalidToken() {
        // Given
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUserId).thenReturn(null);
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUser).thenReturn(null);

        when(oauth2TokenApi.checkAccessToken("invalid-token"))
                .thenReturn(CommonResult.success(null));

        // When
        Boolean result = tokenService.verifyToken("invalid-token");

        // Then
        assertFalse(result);
    }

    @Test
    void testGetUsernameWithExistingUser() {
        // Given
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUserId).thenReturn(12345L);

        // When
        String username = tokenService.getUsername("test-token");

        // Then
        assertEquals("12345", username);
    }

    @Test
    void testGetRolesForAdmin() {
        // Given

        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUser).thenReturn(loginUser);
        securityUtilsMock.when(SecurityFrameworkUtils::getLoginUserId).thenReturn(1L);

        // Then

        assertArrayEquals(new String[]{"admin"}, roles);
    }

    @Test

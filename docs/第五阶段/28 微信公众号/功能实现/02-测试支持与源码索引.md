# 02-测试支持与源码索引

## 测试支持

### 1. 测试策略概述

微信公众号模块的测试面临一个核心挑战：**依赖微信服务器的外部 API**。为了解决这个问题，我们采用以下策略：

1. **Mock WxMpService**：模拟微信 API 的响应
2. **Mock WxMpMessageRouter**：模拟消息路由
3. **集成测试**：使用内存数据库验证业务逻辑

    private MpAccountService mpAccountService;

        String appId = "wx_test_appid";
        String xmlContent = "<xml>" +
                "<ToUserName>gh_test</ToUserName>" +
                "<FromUserName>testOpenId</FromUserName>" +
                "<CreateTime>1234567890</CreateTime>" +
                "<MsgType>text</MsgType>" +
                "<Content>你好</Content>" +
                "<MsgId>123456</MsgId>" +
                "</xml>";

        // 模拟公众号账号
        MpAccountDO account = new MpAccountDO();
        account.setId(1L);
        account.setAppId(appId);
        account.setTenantId(1L);
        when(mpAccountService.getAccountFromCache(appId)).thenReturn(account);

        // 模拟 WxMpService
        WxMpService wxMpService = org.mockito.Mockito.mock(WxMpService.class);
        when(wxMpService.checkSignature(any(), any(), any())).thenReturn(true);
        when(mpServiceFactory.getRequiredMpService(appId)).thenReturn(wxMpService);

        // 模拟消息路由器
        WxMpMessageRouter messageRouter = org.mockito.Mockito.mock(WxMpMessageRouter.class);
        WxMpXmlOutMessage outMessage = WxMpXmlOutMessage.TEXT()
                .content("欢迎！")
                .fromUser("gh_test")
                .toUser("testOpenId")
                .build();
        when(messageRouter.route(any(WxMpXmlMessage.class))).thenReturn(outMessage);
        when(mpServiceFactory.getRequiredMpMessageRouter(appId)).thenReturn(messageRouter);

        // When & Then
        mockMvc.perform(post("/mp/open/{appId}", appId)
                        .contentType("application/xml")
                        .content(xmlContent)
                        .param("timestamp", "1234567890")
                        .param("nonce", "testNonce")
                        .param("signature", "testSignature"))
                .andExpect(status().isOk())
                .andExpect(content().string(org.hamcrest.Matchers.containsString("欢迎！")));
    }
}

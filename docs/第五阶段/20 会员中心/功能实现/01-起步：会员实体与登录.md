# 01 - 起步：会员实体与登录

## 本章目标

从零搭建会员模块的数据层和认证层。完成后，你将拥有：

1. 会员用户核心实体 `MemberUserDO`
2. 四种登录方式的实现（手机号密码、短信验证码、社交登录、微信小程序）
3. 复用 system 模块的 OAuth2 令牌体系
4. 收货地址管理

## 一、核心实体设计

### 1.1 MemberUserDO -- 会员用户

会员用户是会员模块的核心实体，与 Admin 用户分离，面向 C 端消费者。

```java
// 来源：yudao-module-member-server/.../dal/dataobject/user/MemberUserDO.java
@TableName(value = "member_user", autoResultMap = true)
public class MemberUserDO extends TenantBaseDO {

    // ========== 账号信息 ==========
    @TableId
    private Long id;
    private String mobile;           // 手机号（登录标识）
    private String password;         // 加密后的密码（BCrypt）
    private Integer status;          // 帐号状态（启用/禁用）
    private String registerIp;       // 注册 IP
    private Integer registerTerminal; // 注册终端
    private String loginIp;          // 最后登录 IP
    private LocalDateTime loginDate;  // 最后登录时间

    // ========== 基础信息 ==========
    private String nickname;         // 用户昵称
    private String avatar;           // 用户头像
    private String name;             // 真实名字
    private Integer sex;             // 性别
    private LocalDateTime birthday;  // 出生日期
    private Integer areaId;          // 所在地区
    private String mark;             // 用户备注

    // ========== 会员体系 ==========
    private Integer point;           // 积分
    private List<Long> tagIds;       // 会员标签列表
    private Long levelId;            // 会员等级编号
    private Integer experience;      // 会员经验
    private Long groupId;            // 会员分组编号
}
```

设计要点：
- **手机号为主标识**：与 Admin 用户的 `username` 不同，会员以 `mobile` 作为登录标识，符合 C 端用户习惯。
- **BCrypt 加密**：密码使用 `BCryptPasswordEncoder` 加密，无需单独存储 salt。
- **积分与经验分离**：`point` 是积分（可消费），`experience` 是经验（用于升级），两者用途不同。
- **标签使用 JSON 数组**：`tagIds` 使用 `LongListTypeHandler` 存储为 JSON 数组，避免额外的关联表。

### 2.2 手机号 + 密码登录

```java
// 来源：yudao-module-member-server/.../service/auth/MemberAuthServiceImpl.java
@Override
public AppAuthLoginRespVO login(AppAuthLoginReqVO reqVO) {
    // 1. 校验手机号和密码
    MemberUserDO user = login0(reqVO.getMobile(), reqVO.getPassword());

    // 2. 如果 socialType 非空，绑定社交用户
    String openid = null;
    if (reqVO.getSocialType() != null) {
        openid = socialUserApi.bindSocialUser(new SocialUserBindReqDTO(
                user.getId(), getUserType().getValue(),
                reqVO.getSocialType(), reqVO.getSocialCode(), reqVO.getSocialState()))
                .getCheckedData();
    }

    // 3. 创建 Token 令牌，记录登录日志
    return createTokenAfterLoginSuccess(user, reqVO.getMobile(),
            LoginLogTypeEnum.LOGIN_MOBILE, openid);
}

private MemberUserDO login0(String mobile, String password) {
    // 校验账号是否存在
    MemberUserDO user = userService.getUserByMobile(mobile);
    if (user == null) {
        createLoginLog(null, mobile, LoginLogTypeEnum.LOGIN_MOBILE, LoginResultEnum.BAD_CREDENTIALS);
        throw exception(AUTH_LOGIN_BAD_CREDENTIALS);
    }
    // 校验密码
    if (!userService.isPasswordMatch(password, user.getPassword())) {
        createLoginLog(user.getId(), mobile, LoginLogTypeEnum.LOGIN_MOBILE, LoginResultEnum.BAD_CREDENTIALS);
        throw exception(AUTH_LOGIN_BAD_CREDENTIALS);
    }
    // 校验是否禁用
    if (CommonStatusEnum.isDisable(user.getStatus())) {
        createLoginLog(user.getId(), mobile, LoginLogTypeEnum.LOGIN_MOBILE, LoginResultEnum.USER_DISABLED);
        throw exception(AUTH_LOGIN_USER_DISABLED);
    }
    return user;
}
```

### 2.3 短信验证码登录

```java
@Override
@Transactional
public AppAuthLoginRespVO smsLogin(AppAuthSmsLoginReqVO reqVO) {
    // 1. 校验验证码
    String userIp = getClientIP();
    smsCodeApi.useSmsCode(AuthConvert.INSTANCE.convert(reqVO,
            SmsSceneEnum.MEMBER_LOGIN.getScene(), userIp)).checkError();

    // 2. 获取或创建用户（不存在则自动注册）
    MemberUserDO user = userService.createUserIfAbsent(reqVO.getMobile(), userIp, getTerminal());
    Assert.notNull(user, "获取用户失败，结果为空");

    // 3. 校验是否禁用
    if (CommonStatusEnum.isDisable(user.getStatus())) {
        createLoginLog(user.getId(), reqVO.getMobile(),
                LoginLogTypeEnum.LOGIN_SMS, LoginResultEnum.USER_DISABLED);
        throw exception(AUTH_LOGIN_USER_DISABLED);
    }

    // 4. 绑定社交用户（如果有）
    String openid = null;
    if (reqVO.getSocialType() != null) {
        openid = socialUserApi.bindSocialUser(new SocialUserBindReqDTO(
                user.getId(), getUserType().getValue(),
                reqVO.getSocialType(), reqVO.getSocialCode(), reqVO.getSocialState()))
                .getCheckedData();
    }

    // 5. 创建 Token
    return createTokenAfterLoginSuccess(user, reqVO.getMobile(),
            LoginLogTypeEnum.LOGIN_SMS, openid);
}
```

设计要点：
- **自动注册**：`createUserIfAbsent()` 在用户不存在时自动创建，实现"一键登录"体验。
- **验证码校验**：通过 `SmsCodeApi` 调用 system 模块的短信服务校验验证码。

### 2.4 社交登录

```java
@Override
@Transactional
public AppAuthLoginRespVO socialLogin(AppAuthSocialLoginReqVO reqVO) {
    // 1. 使用 code 授权码获取社交用户信息
    SocialUserRespDTO socialUser = socialUserApi.getSocialUserByCode(
            UserTypeEnum.MEMBER.getValue(), reqVO.getType(),
            reqVO.getCode(), reqVO.getState()).getCheckedData();
    if (socialUser == null) {
        throw exception(AUTH_SOCIAL_USER_NOT_FOUND);
    }

    // 2. 获取或创建会员用户
    MemberUserDO user;
    if (socialUser.getUserId() != null) {
        // 已绑定，直接读取用户信息
        user = userService.getUser(socialUser.getUserId());
    } else {
        // 未绑定，注册用户 + 绑定社交账号
        user = userService.createUser(socialUser.getNickname(), socialUser.getAvatar(),
                getClientIP(), getTerminal());
        socialUserApi.bindSocialUser(new SocialUserBindReqDTO(
                user.getId(), getUserType().getValue(),
                reqVO.getType(), reqVO.getCode(), reqVO.getState())).checkError();
    }
    if (user == null) {
        throw exception(USER_NOT_EXISTS);
    }

    // 3. 创建 Token
    return createTokenAfterLoginSuccess(user, user.getMobile(),
            LoginLogTypeEnum.LOGIN_SOCIAL, socialUser.getOpenid());
}
```

### 2.5 微信小程序一键登录

```java
@Override
public AppAuthLoginRespVO weixinMiniAppLogin(AppAuthWeixinMiniAppLoginReqVO reqVO) {
    // 1. 获取手机号信息
    SocialWxPhoneNumberInfoRespDTO phoneNumberInfo = socialClientApi.getWxMaPhoneNumberInfo(
            UserTypeEnum.MEMBER.getValue(), reqVO.getPhoneCode()).getCheckedData();
    Assert.notNull(phoneNumberInfo, "获得手机信息失败，结果为空");

    // 2. 获取或创建用户
    MemberUserDO user = userService.createUserIfAbsent(
            phoneNumberInfo.getPurePhoneNumber(),
            getClientIP(), TerminalEnum.WECHAT_MINI_PROGRAM.getTerminal());
    Assert.notNull(user, "获取用户失败，结果为空");

    // 3. 绑定社交用户
    String openid = socialUserApi.bindSocialUser(new SocialUserBindReqDTO(
            user.getId(), getUserType().getValue(),
            SocialTypeEnum.WECHAT_MINI_PROGRAM.getType(),
            reqVO.getLoginCode(), reqVO.getState())).getCheckedData();

    // 4. 创建 Token
    return createTokenAfterLoginSuccess(user, user.getMobile(),
            LoginLogTypeEnum.LOGIN_SOCIAL, openid);
}
```

设计要点：
- **手机号获取**：微信小程序通过 `phoneCode` 获取用户手机号，无需用户手动输入。
- **自动注册**：手机号不存在时自动创建会员账号。

## 三、OAuth2 令牌体系

会员模块复用 system 模块的 OAuth2 令牌体系，通过 `OAuth2TokenCommonApi` 接口创建和管理令牌。

### 3.1 创建令牌

```java
private AppAuthLoginRespVO createTokenAfterLoginSuccess(MemberUserDO user, String mobile,
                                                        LoginLogTypeEnum logType, String openid) {
    // 1. 插入登录日志
    createLoginLog(user.getId(), mobile, logType, LoginResultEnum.SUCCESS);

    // 2. 创建 Token 令牌
    OAuth2AccessTokenRespDTO accessTokenRespDTO = oauth2TokenApi.createAccessToken(
            new OAuth2AccessTokenCreateReqDTO()
                    .setUserId(user.getId())
                    .setUserType(getUserType().getValue())
                    .setClientId(OAuth2ClientConstants.CLIENT_ID_DEFAULT))
            .getCheckedData();

    // 3. 构建返回结果
    return AuthConvert.INSTANCE.convert(accessTokenRespDTO, openid);
}
```

设计要点：
- **用户类型区分**：`UserTypeEnum.MEMBER` 与 `UserTypeEnum.ADMIN` 区分会员和管理员的令牌。
- **客户端标识**：使用 `CLIENT_ID_DEFAULT` 作为默认客户端，支持多端登录控制。

### 3.2 登出

```java
@Override
public void logout(String token) {
    // 删除访问令牌
    OAuth2AccessTokenRespDTO accessTokenRespDTO = oauth2TokenApi.removeAccessToken(token)
            .getCheckedData();
    if (accessTokenRespDTO == null) {
        return;
    }
    // 记录登出日志
    createLogoutLog(accessTokenRespDTO.getUserId());
}
```

### 3.3 刷新令牌

```java
@Override
public AppAuthLoginRespVO refreshToken(String refreshToken) {
    OAuth2AccessTokenRespDTO accessTokenDO = oauth2TokenApi.refreshAccessToken(
            refreshToken, OAuth2ClientConstants.CLIENT_ID_DEFAULT).getCheckedData();
    return AuthConvert.INSTANCE.convert(accessTokenDO, null);
}
```

```java
// 来源：yudao-module-member-server/.../service/address/AddressServiceImpl.java
@Override
public Long createAddress(Long userId, AddressCreateReqVO reqVO) {
    // 如果是默认地址，先取消其他默认地址
    if (BooleanUtil.isTrue(reqVO.getDefaultStatus())) {
        cancelDefaultAddress(userId);
    }
    // 插入地址
    AddressDO address = AddressConvert.INSTANCE.convert(reqVO).setUserId(userId);
    addressMapper.insert(address);
    return address.getId();
}

private void cancelDefaultAddress(Long userId) {
    AddressDO address = addressMapper.selectByUserIdAndDefaultStatus(userId, true);
    if (address != null) {
        addressMapper.updateById(new AddressDO().setId(address.getId()).setDefaultStatus(false));
    }
}
```

## 六、与 system 模块的集成

会员模块依赖 system 模块提供以下能力：

| 能力 | API | 说明 |
|------|-----|------|
| 短信验证码 | `SmsCodeApi` | 发送和校验短信验证码 |
| 社交登录 | `SocialUserApi` | 社交用户绑定和查询 |
| 社交客户端 | `SocialClientApi` | 获取社交登录 URL、微信手机号 |
| OAuth2 令牌 | `OAuth2TokenCommonApi` | 创建、刷新、删除令牌 |
| 登录日志 | `LoginLogApi` | 记录登录/登出日志 |

这些依赖通过 Feign RPC 调用，会员模块不需要自己实现这些能力。

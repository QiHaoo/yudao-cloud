# 03 - 测试支持与源码索引

## 一、测试支持

### 1.1 测试基础设施

会员模块的测试依赖以下基础设施：

1. **H2 内存数据库**：单元测试使用 H2 内存数据库，无需依赖外部 MySQL。
2. **Mock 依赖**：使用 Mockito mock 外部依赖（如 `SmsCodeApi`、`SocialUserApi`）。
3. **Spring Boot Test**：使用 `@SpringBootTest` 注解加载完整的 Spring 上下文。

### 1.2 常见测试场景

#### 场景一：手机号密码登录测试

```java
@SpringBootTest
class MemberAuthServiceTest {

    @Resource
    private MemberAuthService authService;

    @Test
    public void testLogin() {
        // 1. 先注册一个用户
        MemberUserDO user = createTestUser("13800138000", "password123");

        // 2. 登录
        AppAuthLoginReqVO reqVO = new AppAuthLoginReqVO()
                .setMobile("13800138000")
                .setPassword("password123");
        AppAuthLoginRespVO respVO = authService.login(reqVO);

        // 3. 验证返回的令牌
        assertNotNull(respVO.getAccessToken());
        assertNotNull(respVO.getRefreshToken());
    }

    @Test
    public void testLoginWithWrongPassword() {
        // 1. 先注册一个用户
        createTestUser("13800138000", "password123");

        // 2. 使用错误密码登录
        AppAuthLoginReqVO reqVO = new AppAuthLoginReqVO()
                .setMobile("13800138000")
                .setPassword("wrong_password");

        // 3. 验证抛出异常
        assertThrows(ServiceException.class, () -> authService.login(reqVO));
    }
}
```

#### 场景二：签到流程测试

```java
@SpringBootTest
class MemberSignInRecordServiceTest {

    @Resource
    private MemberSignInRecordService signInService;

    @Test
    public void testCreateSignRecord() {
        Long userId = 1L;

        // 1. 签到
        MemberSignInRecordDO record = signInService.createSignRecord(userId);

        // 2. 验证签到记录
        assertNotNull(record);
        assertEquals(1, record.getDay()); // 第一天签到
        assertTrue(record.getPoint() > 0); // 获得积分
    }

    @Test
    public void testDuplicateSignIn() {
        Long userId = 1L;

        // 1. 第一次签到
        signInService.createSignRecord(userId);

        // 2. 第二次签到（应该抛出异常）
        assertThrows(ServiceException.class, () -> signInService.createSignRecord(userId));
    }
}
```

#### 场景三：积分变动测试

```java
@SpringBootTest
class MemberPointRecordServiceTest {

    @Resource
    private MemberPointRecordService pointService;

    @Resource
    private MemberUserService userService;

    @Test
    public void testAddPoint() {
        Long userId = 1L;

        // 1. 增加积分
        pointService.createPointRecord(userId, 100,
                MemberPointBizTypeEnum.SIGN, "1");

        // 2. 验证用户积分
        MemberUserDO user = userService.getUser(userId);
        assertEquals(100, user.getPoint());
    }

    @Test
    public void testSubPointInsufficient() {
        Long userId = 1L;

        // 1. 尝试扣减积分（余额不足）
        assertThrows(ServiceException.class, () ->
                pointService.createPointRecord(userId, -100,
                        MemberPointBizTypeEnum.REFUND, "1"));
    }
}
```

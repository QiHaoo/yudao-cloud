# 03 - 精简汇总

## MES 流程速查

### 生产全流程

```
ERP 工单 → MES 工单 → BOM 展开 → 领料 → 排产(任务) → 生产(报工) → 质检 → 入库
```

### 生产执行链路

```
工单(WorkOrder) → 任务(Task) → 报工(WorkRecord) → 更新工单进度
                      │
                      ├── 关联工序(Process)
                      └── 关联工作站(Workstation)
```

### 质量检验时机

| 检验类型 | 时机 | 对象 |
|---------|------|------|
| IQC | 原材料入库前 | 来料 |
| IPQC | 生产过程中 | 在制品 |
| OQC | 成品出库前 | 成品 |
| RQC | 客户退货后 | 退货品 |

### 设备维护类型

| 类型 | 频率 | 目的 |
|------|------|------|
| 点检 (Check) | 每日 | 日常巡检 |
| 保养 (Maintenance) | 定期 | 预防性维护 |
| 维修 (Repair) | 故障时 | 纠正性维护 |

### BOM 展开

```
产品 BOM → 乘以工单数量 → 物料需求清单 → 对比库存 → 生成领料单
```

## 核心设计决策

| 决策 | 选择 | 放弃的方案 | 原因 |
|------|------|-----------|------|
| BOM 管理 | 独立 BOM 表 | 工单内嵌物料列表 | BOM 可复用，多工单共享 |
| 工单拆分 | 工单 → 任务两级 | 单级工单 | 支持并行生产、灵活分配 |
| 报工粒度 | 工序级报工 | 工单级报工 | 精确跟踪每道工序的进度 |
| 质检模板 | 模板 + 指标配置 | 硬编码检验项 | 不同产品可自定义检验标准 |
| 设备管理 | 点检/保养/维修三套 | 统一维护记录 | 不同维护类型有不同的关注点 |
| 车间仓储 | MES 内置 wm 模块 | 只用 WMS | 车间级物料管理更精细 |
| 排班管理 | 日历 + 班组 + 班次 | 简单班次表 | 支持复杂排班需求 |

## 工单状态流转

```
未排产(0) ──排产──> 排产中(10) ──开始──> 生产中(20) ──完成──> 已完成(30)
                       │                    │
                       └── 取消 ──> 已取消(40)
```

## 模块集成关系

```
MES
  │
  ├── ◄── ERP（生产工单来源、物料主数据）
  │
  ├── ◄── System（用户、部门）
  │
  ├── ◄── WMS（物料领用、成品入库）
  │
  ├── ──> IoT（设备数据采集）
  │
  └── ──> 消息通知（质量异常、设备故障）
```

# 03 - 精简汇总

## WMS 流程速查

### 入库流程

```
创建入库单 → 填写明细(SKU+数量) → 审核 → 更新库存 → 记录变动历史
```

### 出库流程

```
创建出库单 → 填写明细 → 库存校验 → 审核 → 更新库存 → 记录变动历史
```

### 盘点流程

```
创建盘点单 → 生成清单 → 实盘录入 → 差异分析 → 审核 → 调整库存
```

### 调拨流程

```
创建调拨单 → 审核 → 源仓库出库 → 目标仓库入库 → 调拨完成
```

### 库存变动来源

| 业务类型 | 变动方向 | 关联单据 |
|---------|---------|---------|
| 采购入库 | +数量 | 入库单 |
| 销售退货 | +数量 | 入库单 |
| 销售出库 | -数量 | 出库单 |
| 采购退货 | -数量 | 出库单 |
| 盘盈 | +数量 | 盘点单 |
| 盘亏 | -数量 | 盘点单 |
| 调拨入库 | +数量 | 调拨单 |
| 调拨出库 | -数量 | 调拨单 |

## 核心设计决策

| 决策 | 选择 | 放弃的方案 | 原因 |
|------|------|-----------|------|
| 库存粒度 | SKU + 仓库 | 产品 + 仓库 | 支持不同规格独立库存 |
| 库存变动 | 独立历史表 | 只存最新值 | 支持追溯和对账 |
| 出库校验 | 创建时预校验 | 审核时校验 | 提前发现问题，减少无效操作 |
| 盘点方式 | 全盘 + 明细 | 只存盈亏结果 | 保留完整盘点记录 |
| 调拨 | 两步（出+入） | 一步完成 | 支持在途库存管理 |
| 入库/出库类型 | 枚举配置 | 硬编码 | 支持扩展新的业务类型 |
| 业务单号 | 冗余存储 | 只存关联 ID | 快速查询，不需要 JOIN |

## WMS 与 ERP 库存对比

| 维度 | WMS | ERP |
|------|-----|-----|
| 粒度 | SKU 级别 | 产品级别 |
| 关注点 | 在哪里、多少个 | 多少钱、对不对 |
| 使用者 | 仓库管理员 | 财务、采购 |
| 变动记录 | 详细历史 | 汇总记录 |

## 模块集成关系

```
WMS
  │
  ├── ◄── ERP（采购入库、销售出库）
  │
  ├── ◄── MES（生产领料、生产入库）
  │
  ├── ◄── Trade（电商订单 → 出库）
  │
  └── ──> 消息通知（库存预警）
```

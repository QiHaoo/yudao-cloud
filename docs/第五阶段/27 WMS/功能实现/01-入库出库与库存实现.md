# 入库出库与库存实现

## 1. 库存管理实现

### 1.1 库存数据模型

WMS 库存的核心数据对象 `WmsInventoryDO` 定义了库存的完整信息：

```java
// 文件位置：yudao-module-wms/yudao-module-wms-server/src/main/java/cn/iocoder/yudao/module/wms/dal/dataobject/inventory/WmsInventoryDO.java

@TableName("wms_inventory")
@Data
@Builder
@NoArgsConstructor
@AllArgsConstructor
public class WmsInventoryDO extends TenantBaseDO {

    @TableId
    private Long id;
    
    /**
     * 仓库编号，关联 WmsWarehouseDO
     */
    private Long warehouseId;
    
    /**
     * 库区编号，关联 WmsWarehouseAreaDO
     */
    private Long areaId;
    
    /**
     * 库位编号，关联 WmsWarehouseLocationDO
     */
    private Long locationId;
    
    /**
     * 物料编号，关联 WmsItemDO
     */
    private Long itemId;
    
    /**
     * SKU 编号，关联 WmsItemSkuDO
     */
    private Long skuId;
    
    /**
     * 批次号
     */
    private String batchNo;
    
    /**
     * 库存数量
     */
    private BigDecimal quantity;
    
    /**
     * 可用数量（库存数量 - 锁定数量）
     */
    private BigDecimal quantityAvailable;
    
    /**
     * 锁定数量（预留给订单等）
     */
    private BigDecimal quantityLocked;
    
    /**
     * 单位，关联 WmsUnitDO
     */
    private Long unitId;
    
    /**
     * 生产日期
     */
    private LocalDateTime productionDate;
    
    /**
     * 过期日期
     */
    private LocalDateTime expirationDate;
    
    /**
     * 库存状态（正常/冻结/待检）
     */
    private Integer status;
}
```

**关键设计说明**：
- 继承 `TenantBaseDO` 支持多租户隔离
- `quantityAvailable = quantity - quantityLocked`，支持库存预留
- 支持批次管理、生产日期、过期日期
- 三级库位结构：仓库 → 库区 → 库位

```java
// 文件位置：yudao-module-wms/yudao-module-wms-server/src/main/java/cn/iocoder/yudao/module/wms/service/inventory/WmsInventoryService.java

public interface WmsInventoryService {

    // ========== 库存查询 ==========
    
    /**
     * 获取库存分页
     */
    PageResult<WmsInventoryDO> getInventoryPage(WmsInventoryPageReqVO pageReqVO);
    
    /**
     * 获取库存列表
     */
    List<WmsInventoryDO> getInventoryList(WmsInventoryListReqVO listReqVO);
    
    /**
     * 获取指定仓库+物料+SKU 的库存
     */
    WmsInventoryDO getInventory(Long warehouseId, Long itemId, Long skuId, String batchNo);
    
    /**
     * 获取物料的总库存数量
     */
    BigDecimal getTotalQuantity(Long itemId, Long skuId);
    
    /**
     * 获取物料的可用库存数量
     */
    BigDecimal getAvailableQuantity(Long itemId, Long skuId);
    
    // ========== 库存变更 ==========
    
    /**
     * 入库（增加库存）
     * 
     * @param warehouseId 仓库编号
     * @param itemId 物料编号
     * @param skuId SKU 编号
     * @param batchNo 批次号
     * @param quantity 入库数量
     * @param sourceType 来源类型（采购入库/生产入库/退货入库）
     * @param sourceId 来源单据编号
     */
    void addInventory(Long warehouseId, Long itemId, Long skuId, String batchNo,
                      BigDecimal quantity, Integer sourceType, Long sourceId);
    
    /**
     * 出库（减少库存）
     * 
     * @param warehouseId 仓库编号
     * @param itemId 物料编号
     * @param skuId SKU 编号
     * @param batchNo 批次号
     * @param quantity 出库数量
     * @param sourceType 来源类型（销售出库/生产领料/报废出库）
     * @param sourceId 来源单据编号
     */
    void subtractInventory(Long warehouseId, Long itemId, Long skuId, String batchNo,
                           BigDecimal quantity, Integer sourceType, Long sourceId);
    
    /**
     * 锁定库存（预留）
     */
    void lockInventory(Long warehouseId, Long itemId, Long skuId, BigDecimal quantity);
    
    /**
     * 解锁库存（释放预留）
     */
    void unlockInventory(Long warehouseId, Long itemId, Long skuId, BigDecimal quantity);
    
    /**
     * 库存转移（库位间转移）
     */
    void transferInventory(Long fromLocationId, Long toLocationId, Long itemId, Long skuId,
                           BigDecimal quantity);
    
    /**
     * 库存盘点（调整库存）
     */
    void adjustInventory(Long inventoryId, BigDecimal actualQuantity, String reason);
}
```

### 1.3 库存控制器层实现

```java
// 文件位置：yudao-module-wms/yudao-module-wms-server/src/main/java/cn/iocoder/yudao/module/wms/controller/admin/inventory/WmsInventoryController.java

@Tag(name = "管理后台 - WMS 库存统计")
@RestController
@RequestMapping("/wms/inventory")
@Validated
public class WmsInventoryController {

    @Resource
    private WmsInventoryService inventoryService;
    @Resource
    private WmsItemService itemService;
    @Resource
    private WmsItemSkuService itemSkuService;
    @Resource
    private WmsWarehouseService warehouseService;

    @GetMapping("/page")
    @Operation(summary = "获得库存统计分页")
    @PreAuthorize("@ss.hasPermission('wms:inventory:query')")
    public CommonResult<PageResult<WmsInventoryRespVO>> getInventoryPage(@Valid WmsInventoryPageReqVO pageReqVO) {
        PageResult<WmsInventoryDO> pageResult = inventoryService.getInventoryPage(pageReqVO);
        return success(new PageResult<>(buildInventoryRespVOList(pageResult.getList()), pageResult.getTotal()));
    }

    @GetMapping("/list")
    @Operation(summary = "获得库存统计列表")
    @PreAuthorize("@ss.hasPermission('wms:inventory:query')")
    public CommonResult<List<WmsInventoryRespVO>> getInventoryList(@Valid WmsInventoryListReqVO listReqVO) {
        List<WmsInventoryDO> list = inventoryService.getInventoryList(listReqVO);
        return success(buildInventoryRespVOList(list));
    }

    // ==================== 拼接 VO ====================

    private List<WmsInventoryRespVO> buildInventoryRespVOList(List<WmsInventoryDO> list) {
        if (CollUtil.isEmpty(list)) {
            return Collections.emptyList();
        }
        Map<Long, WmsItemSkuDO> skuMap = itemSkuService.getItemSkuMap(convertSet(list, WmsInventoryDO::getSkuId));
        Map<Long, WmsItemDO> itemMap = itemService.getItemMap(convertSet(skuMap.values(), WmsItemSkuDO::getItemId));
        Map<Long, WmsWarehouseDO> warehouseMap = warehouseService.getWarehouseMap(
                convertSet(list, WmsInventoryDO::getWarehouseId));
        
        return BeanUtils.toBean(list, WmsInventoryRespVO.class, vo -> {
            MapUtils.findAndThen(skuMap, vo.getSkuId(), sku -> {
                vo.setSkuCode(sku.getCode()).setSkuName(sku.getName()).setItemId(sku.getItemId());
                MapUtils.findAndThen(itemMap, sku.getItemId(), item -> vo.setItemCode(item.getCode())
                        .setItemName(item.getName()).setUnit(item.getUnit()));
            });
            MapUtils.findAndThen(warehouseMap, vo.getWarehouseId(), warehouse -> 
                vo.setWarehouseName(warehouse.getName()));
        });
    }
}
```

## 总结

### 核心实现要点

1. **库存管理**：支持批次管理、库存预留、三级库位结构
2. **入库管理**：支持多种入库类型，自动更新库存
3. **出库管理**：校验库存充足性，自动扣减库存
4. **库存变更历史**：完整记录每次库存变动
5. **盘点管理**：支持全盘/抽盘/动碰盘，自动调整库存
6. **物料管理**：支持 SKU 管理、条形码、库存上下限

### 关键业务流程

| 流程 | 核心步骤 | 关键服务 |
|------|---------|---------|
| 入库 | 创建入库单 → 确认入库 → 增加库存 | `WmsInboundOrderService` + `WmsInventoryService` |
| 出库 | 创建出库单 → 校验库存 → 确认出库 → 减少库存 | `WmsOutboundOrderService` + `WmsInventoryService` |
| 盘点 | 创建盘点单 → 录入结果 → 完成盘点 → 调整库存 | `WmsCheckOrderService` + `WmsInventoryService` |
| 转移 | 选择源/目标库位 → 确认转移 → 更新库存位置 | `WmsInventoryService.transferInventory` |

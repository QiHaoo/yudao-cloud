# 测试支持与源码索引

## 1. 测试支持

### 1.1 测试现状分析

IoT 模块的测试面临以下挑战：

| 挑战 | 说明 | 当前状态 |
|------|------|---------|
| 设备模拟 | 需要模拟真实设备行为 | 手动 Mock |
| 协议测试 | MQTT/CoAP/HTTP 多协议 | 缺少集成测试 |
| 时序数据 | TDengine 依赖 | 本地测试困难 |
| 消息总线 | Redis/RocketMQ 依赖 | 需要外部服务 |

### 1.2 测试工具设计

#### 设备模拟器

```java
/**
 * IoT 设备模拟器
 * 用于单元测试和集成测试
 */
public class IotDeviceSimulator {

    private final String productKey;
    private final String deviceName;
    private final String deviceSecret;

    /**
     * 模拟设备认证
     */
    public boolean simulateAuth() {
        IotDeviceAuthReqDTO reqDTO = new IotDeviceAuthReqDTO();
        reqDTO.setProductKey(productKey);
        reqDTO.setDeviceName(deviceName);
        reqDTO.setDeviceSecret(deviceSecret);
        return deviceCommonApi.authDevice(reqDTO).getData();
    }

    /**
     * 模拟属性上报
     */
    public void simulatePropertyPost(String identifier, Object value) {
        IotDeviceMessage message = new IotDeviceMessage();
        message.setMethod("thing.property.post");
        message.setParams(Collections.singletonMap(identifier, value));
        deviceMessageService.handleUpstreamDeviceMessage(message, getDevice());
    }

    /**
     * 模拟事件上报
     */
    public void simulateEventPost(String eventId, Map<String, Object> params) {
        IotDeviceMessage message = new IotDeviceMessage();
        message.setMethod("thing.event.post");
        message.setId(eventId);
        message.setParams(params);
        deviceMessageService.handleUpstreamDeviceMessage(message, getDevice());
    }
}
```

#### Mock 消息总线

```java
/**
 * 本地 Mock 消息总线
 * 用于单元测试，不依赖外部服务
 */
public class IotMockMessageBus implements IotMessageBus {

    private final Map<String, List<IotMessageSubscriber>> subscribers = new ConcurrentHashMap<>();

    @Override
    public void publish(String topic, IotLocalMessage message) {
        List<IotMessageSubscriber> topicSubscribers = subscribers.get(topic);
        if (topicSubscribers != null) {
            topicSubscribers.forEach(subscriber -> subscriber.onMessage(message));
        }
    }

    @Override
    public void subscribe(String topic, IotMessageSubscriber subscriber) {
        subscribers.computeIfAbsent(topic, k -> new ArrayList<>()).add(subscriber);
    }
}
```

### 1.4 常见测试场景

#### 设备认证测试

```java
@Test
void testDeviceAuth() {
    // 1. 创建测试设备
    IotDeviceDO device = createTestDevice("test-device-001");
    
    // 2. 模拟认证请求
    IotDeviceAuthReqDTO authReqDTO = new IotDeviceAuthReqDTO();
    authReqDTO.setProductKey("test-product-key");
    authReqDTO.setDeviceName("test-device-001");
    authReqDTO.setDeviceSecret(device.getDeviceSecret());
    
    // 3. 执行认证
    boolean result = deviceService.authDevice(authReqDTO);
    
    // 4. 验证结果
    assertTrue(result);
}
```

#### 设备消息测试

```java
@Test
void testDevicePropertyPost() {
    // 1. 创建测试设备
    IotDeviceDO device = createTestDevice("test-device-002");
    
    // 2. 构造属性上报消息
    IotDeviceMessage message = new IotDeviceMessage();
    message.setMethod("thing.property.post");
    message.setDeviceId(device.getId());
    message.setParams(Map.of("temperature", 25.5, "humidity", 60));
    
    // 3. 处理消息
    deviceMessageService.handleUpstreamDeviceMessage(message, device);
    
    // 4. 验证属性更新
    IotDevicePropertyDO property = devicePropertyService
        .getDeviceProperty(device.getId(), "temperature");
    assertEquals(25.5, property.getValue());
}
```

#### 网关拓扑测试

```java
@Test
void testGatewayTopology() {
    // 1. 创建网关设备
    IotDeviceDO gateway = createTestDevice("gateway-001");
    gateway.setDeviceType(DeviceTypeEnum.GATEWAY.getType());
    deviceService.updateDevice(gateway);
    
    // 2. 创建子设备
    IotDeviceDO subDevice = createTestDevice("sub-device-001");
    subDevice.setDeviceType(DeviceTypeEnum.DEVICE.getType());
    deviceService.updateDevice(subDevice);
    
    // 3. 绑定子设备到网关
    deviceService.bindDeviceGateway(
        Collections.singletonList(subDevice.getId()), 
        gateway.getId()
    );
    
    // 4. 验证拓扑关系
    List<IotDeviceDO> subDevices = deviceService.getDeviceListByGatewayId(gateway.getId());
    assertEquals(1, subDevices.size());
    assertEquals(subDevice.getId(), subDevices.get(0).getId());
}
```

### 2.3 业务层使用示例

#### 创建设备

```java
@Resource
private IotDeviceService deviceService;

public Long createDevice() {
    IotDeviceSaveReqVO reqVO = new IotDeviceSaveReqVO();
    reqVO.setDeviceName("温度传感器001");
    reqVO.setProductId(1L);
    reqVO.setDeviceType(DeviceTypeEnum.DEVICE.getType());
    reqVO.setNickname("车间温度传感器");
    return deviceService.createDevice(reqVO);
}
```

#### 发送设备消息

```java
@Resource
private IotDeviceMessageService deviceMessageService;

public void sendCommandToDevice(Long deviceId, String command) {
    IotDeviceMessage message = new IotDeviceMessage();
    message.setMethod("thing.service.request");
    message.setDeviceId(deviceId);
    message.setParams(Map.of("service", "switch", "command", command));
    
    IotDeviceDO device = deviceService.getDevice(deviceId);
    deviceMessageService.sendDeviceMessage(message, device);
}
```

#### 查询设备属性

```java
@Resource
private IotDevicePropertyService devicePropertyService;

public Map<String, Object> getDeviceProperties(Long deviceId) {
    List<IotDevicePropertyDO> properties = devicePropertyService.getDevicePropertyList(deviceId);
    return properties.stream()
        .collect(Collectors.toMap(
            IotDevicePropertyDO::getIdentifier,
            IotDevicePropertyDO::getValue
        ));
}
```

### 2.6 核心设计模式

| 模式 | 应用场景 | 示例 |
|------|---------|------|
| **策略模式** | 多协议支持 | `IotProtocol` 接口 + 多实现 |
| **观察者模式** | 消息总线 | `IotMessageBus` + `IotMessageSubscriber` |
| **工厂模式** | 消息创建 | `IotDeviceMessage` 构建 |
| **模板方法** | 协议处理器 | `AbstractIotProtocolDownstreamSubscriber` |
| **代理模式** | 设备缓存 | `getDeviceFromCache` 方法 |

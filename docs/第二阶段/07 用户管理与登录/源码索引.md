# 源码索引：用户管理与登录

## 源码文件列表

所有路径相对于 `yudao-module-system/yudao-module-system-server/src/main/java/cn/iocoder/yudao/module/system/`

### Controller 层

| 文件 | 路径 | 说明 |
|------|------|------|
| ★ AuthController | `controller/admin/auth/AuthController.java` | 认证端点：登录/登出/刷新/权限信息 |
| UserController | `controller/admin/user/UserController.java` | 用户管理端点：CRUD/导入导出 |

### 请求/响应 VO

| 文件 | 路径 | 说明 |
|------|------|------|
| AuthLoginReqVO | `controller/admin/auth/vo/AuthLoginReqVO.java` | 登录请求（用户名+密码+验证码） |
| AuthLoginRespVO | `controller/admin/auth/vo/AuthLoginRespVO.java` | 登录响应（access_token + refresh_token） |
| AuthPermissionInfoRespVO | `controller/admin/auth/vo/AuthPermissionInfoRespVO.java` | 权限信息响应（用户+角色+权限集合） |

### Service 层

| 文件 | 路径 | 说明 |
|------|------|------|
| ★ AdminAuthServiceImpl | `service/auth/AdminAuthServiceImpl.java` | 管理后台登录业务：密码校验、Token 创建 |
| ★ OAuth2TokenServiceImpl | `service/oauth2/OAuth2TokenServiceImpl.java` | Token 管理：创建/刷新/校验/过期处理 |
| OAuth2ClientServiceImpl | `service/oauth2/OAuth2ClientServiceImpl.java` | OAuth2 客户端管理 |
| OAuth2GrantServiceImpl | `service/oauth2/OAuth2GrantServiceImpl.java` | 授权模式处理：密码/刷新/短信/社交 |
| AdminUserServiceImpl | `service/user/AdminUserServiceImpl.java` | 用户 CRUD：创建/修改/删除/导入导出 |

### 数据访问层（DO 实体）

| 文件 | 路径 | 说明 |
|------|------|------|
| ★ AdminUserDO | `dal/dataobject/user/AdminUserDO.java` | 用户实体：账号、密码、部门、状态 |
| ★ OAuth2AccessTokenDO | `dal/dataobject/oauth2/OAuth2AccessTokenDO.java` | 访问令牌实体：用户ID、客户端ID、过期时间 |
| OAuth2RefreshTokenDO | `dal/dataobject/oauth2/OAuth2RefreshTokenDO.java` | 刷新令牌实体：关联访问令牌 |
| OAuth2ClientDO | `dal/dataobject/oauth2/OAuth2ClientDO.java` | OAuth2 客户端：client_id、secret、授权范围 |

### 数据访问层（Redis）

| 文件 | 路径 | 说明 |
|------|------|------|
| ★ OAuth2AccessTokenRedisDAO | `dal/redis/oauth2/OAuth2AccessTokenRedisDAO.java` | Token Redis 存储：读写/删除/过期管理 |

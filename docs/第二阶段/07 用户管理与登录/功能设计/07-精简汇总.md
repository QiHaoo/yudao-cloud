# 07-精简汇总

## 能力与使用速查

### 登录方式

| 方式 | 端点 | 参数 | 说明 |
|------|------|------|------|
| 账号密码 | POST /system/auth/login | username, password, captchaVerification | 默认方式 |
| 短信验证码 | POST /system/auth/sms-login | mobile, code | 需先调用 send-sms-code |
| 社交登录 | POST /system/auth/social-login | type, code, state | 微信/钉钉等 |
| 刷新令牌 | POST /system/auth/refresh-token | refreshToken | Token 快过期时调用 |
| 登出 | POST /system/auth/logout | 无 | 从 Header 提取 Token |

### 用户管理

| 操作 | 端点 | 权限标识 |
|------|------|---------|
| 创建用户 | POST /system/user/create | system:user:create |
| 修改用户 | PUT /system/user/update | system:user:update |
| 删除用户 | DELETE /system/user/delete | system:user:delete |
| 查询用户 | GET /system/user/get | system:user:query |
| 分页查询 | GET /system/user/page | system:user:query |
| 修改密码 | PUT /system/user/update-password | system:user:update-password |
| 启用/禁用 | PUT /system/user/update-status | system:user:update |

### Token 生命周期

```
登录 → 创建 AccessToken + RefreshToken
  │
  ├─ AccessToken 有效 → 正常使用
  │
  ├─ AccessToken 过期 → 用 RefreshToken 刷新 → 创建新 AccessToken
  │
  ├─ RefreshToken 过期 → 重新登录
  │
  └─ 登出 → 删除 AccessToken + RefreshToken
```

## 设计与概念精要

### 核心设计决策

1. **UUID Token + Redis**：选择可撤销的 Token 方案，而非无状态的 JWT
2. **双写存储**：Redis 提供快速查询，MySQL 提供审计持久化
3. **可配置验证码**：生产开启、开发关闭，通过配置切换
4. **统一登录入口**：所有登录方式最终调用 `createTokenAfterLoginSuccess()`
5. **密码安全**：BCrypt 单向加密，统一错误信息防止用户枚举

### 核心实体

| 实体 | 说明 | 关键字段 |
|------|------|---------|
| AdminUserDO | 管理后台用户 | username, password, deptId, status |
| OAuth2AccessTokenDO | 访问令牌 | accessToken, refreshToken, userId, expiresTime |
| OAuth2RefreshTokenDO | 刷新令牌 | refreshToken, userId, expiresTime |
| OAuth2ClientDO | OAuth2 客户端 | clientId, secret, accessTokenValiditySeconds |

### 关键类

| 类 | 职责 |
|----|------|
| AuthController | 认证 HTTP 端点 |
| AdminAuthServiceImpl | 登录/登出/注册业务逻辑 |
| OAuth2TokenServiceImpl | Token 创建/刷新/删除 |
| OAuth2AccessTokenRedisDAO | Token 的 Redis 存储 |

### 与组件 6 的关系

```
组件 6：验证 Token（"你是谁"）→ TokenAuthenticationFilter
组件 7：创建 Token（"给你证件"）→ AdminAuthServiceImpl + OAuth2TokenServiceImpl
```

## 配置文件

```yaml
# application.yml 中的多租户配置
yudao:
  tenant:
    enable: true                          # 全局开关
    ignore-urls:                          # 豁免 URL
      - /admin-api/pay/callback/**
    ignore-visit-urls:                    # 不允许跨租户访问的 URL
      - /admin-api/user/profile/**
    ignore-tables:                        # 豁免的表
      - system_tenant
      - system_dict_type
      - system_dict_data
    ignore-caches:                        # 豁免的缓存
      - dict_data
      - system_config
```

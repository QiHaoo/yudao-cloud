## 二、业务层使用示例

### 2.1 文件上传使用示例

```java
// 在业务 Controller 中使用文件上传
@RestController
@RequestMapping("/product/spu")
public class ProductSpuController {

    @Resource
    private FileService fileService;

    @PostMapping("/upload-image")
    @Operation(summary = "上传商品图片")
    public CommonResult<String> uploadImage(@RequestParam("file") MultipartFile file) throws Exception {
        byte[] content = IoUtil.readBytes(file.getInputStream());
        // 调用文件服务上传，指定目录为 product
        return success(fileService.createFile(content, file.getOriginalFilename(),
                "product", file.getContentType()));
    }
}
```

### 2.2 代码生成使用示例

```java
// 在管理后台调用代码生成接口
// 1. 获取数据库表列表
GET /infra/codegen/db/table/list?dataSourceConfigId=1&name=product

// 2. 导入表到代码生成器
POST /infra/codegen/create-list
{
    "dataSourceConfigId": 1,
    "tableNames": ["product_spu", "product_sku"]
}

// 3. 修改生成配置（可选）
PUT /infra/codegen/update-table
{
    "id": 1,
    "scene": 2,
    "templateType": 1,
    "frontType": 3
}

// 4. 预览生成代码
GET /infra/codegen/preview?id=1

// 5. 下载生成代码
GET /infra/codegen/download?id=1
```

## 三、配置文件

### 3.1 代码生成配置

```yaml
# application-dev.yml
yudao:
  codegen:
    base-package: cn.iocoder.yudao.module.product  # 生成代码的基础包
    db-schemas: yudao_product                       # 数据库名
    front-type: 3                                    # 前端类型（Vue3 Element Plus）
    vo-type: 1                                       # VO 类型
    delete-batch-enable: true                        # 是否生成批量删除
    unit-test-enable: true                           # 是否生成单元测试
    import-enable: true                              # 是否生成 Excel 导入
```

### 3.2 文件存储配置

文件存储配置存储在数据库 `infra_file_config` 表中，支持多配置：

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 配置编号 | 1 |
| name | 配置名称 | 阿里云 OSS |
| storage | 存储器类型 | 20 (S3) |
| master | 是否主配置 | true |
| config | JSON 配置 | `{"endpoint":"oss-cn-hangzhou.aliyuncs.com","accessKey":"...","bucket":"yudao"}` |

### CodegenServiceImpl 测试

源码位置：`yudao-module-infra-server/src/test/java/cn/iocoder/yudao/module/infra/service/codegen/CodegenServiceImplTest.java`

```java
@ExtendWith(MockitoExtension.class)
class CodegenServiceImplTest {

    @InjectMocks
    private CodegenServiceImpl codegenService;

    @Mock
    private DatabaseTableService databaseTableService;
    @Mock
    private CodegenTableMapper codegenTableMapper;
    @Mock
    private CodegenColumnMapper codegenColumnMapper;
    @Mock
    private CodegenBuilder codegenBuilder;
    @Mock
    private CodegenEngine codegenEngine;

    @Test
    @DisplayName("测试导入表结构")
    void testCreateCodegenList() {
        // 准备参数
        Long dataSourceConfigId = 1L;
        String tableName = "product_spu";
        CodegenCreateListReqVO reqVO = new CodegenCreateListReqVO()
                .setDataSourceConfigId(dataSourceConfigId)
                .setTableNames(Collections.singletonList(tableName));

        // Mock 数据库表结构
        TableInfo tableInfo = new TableInfo();
        tableInfo.setName(tableName);
        tableInfo.setComment("商品 SPU 表");
        when(databaseTableService.getTable(dataSourceConfigId, tableName))
                .thenReturn(tableInfo);

        // Mock CodegenBuilder
        CodegenTableDO tableDO = new CodegenTableDO();
        tableDO.setId(1L);
        when(codegenBuilder.buildTable(tableInfo)).thenReturn(tableDO);
        when(codegenBuilder.buildColumns(eq(1L), anyList()))
                .thenReturn(Collections.emptyList());

        // 执行测试
        List<Long> ids = codegenService.createCodegenList("admin", reqVO);

        // 验证结果
        assertThat(ids).hasSize(1);
        verify(codegenTableMapper).insert(any(CodegenTableDO.class));
    }

    @Test
    @DisplayName("测试校验表信息 - 表注释为空")
    void testValidateTableInfo_TableCommentIsNull() {
        // 准备参数
        TableInfo tableInfo = new TableInfo();
        tableInfo.setName("test_table");
        tableInfo.setComment("");  // 空注释

        // 执行测试
        assertThatThrownBy(() -> codegenService.validateTableInfo(tableInfo))
                .hasMessage(CODEGEN_TABLE_INFO_TABLE_COMMENT_IS_NULL.toString());
    }
}
```

**测试要点**：
- 使用 `@ExtendWith(MockitoExtension.class)` 启用 Mockito
- 使用 `@InjectMocks` 注入被测试对象
- 使用 `@Mock` 创建依赖的 Mock 对象
- 使用 `verify()` 验证方法调用

## 常见问题

### Q1: 代码生成后如何调整模板？

A: 修改 `yudao-module-infra-server/src/main/resources/codegen/` 目录下的 `.vm` 模板文件，重启应用即可生效。

### Q2: 如何添加新的存储类型？

A: 实现 `FileClient` 接口，在 `FileStorageEnum` 中添加枚举值，在 `FileClientFactoryImpl` 中注册新客户端。

### Q3: 操作日志如何异步写入？

A: `ApiAccessLogFilter` 通过 `ApiAccessLogCommonApi.createApiAccessLogAsync()` 异步写入，该方法使用 `@Async` 注解。

### Q4: 如何自定义脱敏字段？

A: 在 `@ApiAccessLog` 注解的 `sanitizeKeys` 属性中添加需要脱敏的字段名：

```java
@ApiAccessLog(sanitizeKeys = {"secretKey", "apiKey"})
```

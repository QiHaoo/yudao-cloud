## 配置文件

```yaml
# application.yml 中的定时任务配置
xxl:
  job:
    enabled: true                                    # 是否开启
    accessToken: default_token                       # 访问令牌
    admin:
      addresses: http://127.0.0.1:9090/xxl-job-admin # 调度中心地址
    executor:
      appname: ${spring.application.name}            # 执行器名称
      logpath: ${user.home}/logs/xxl-job/${spring.application.name}  # 日志路径
      # port: -1                                     # 端口（默认随机）
      # log-retention-days: 30                       # 日志保留天数（默认 30）
```

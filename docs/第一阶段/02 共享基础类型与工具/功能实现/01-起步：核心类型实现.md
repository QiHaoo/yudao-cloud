## 1. 模块 pom.xml

`yudao-common` 是一个纯定义 + 工具模块，不包含 Spring 自动配置。它的依赖设计原则是：**大部分运行时依赖用 `provided` scope**。

```xml
<!-- yudao-framework/yudao-common/pom.xml -->
<project>
    <parent>
        <groupId>cn.iocoder.cloud</groupId>
        <artifactId>yudao-framework</artifactId>
        <version>${revision}</version>
    </parent>

    <artifactId>yudao-common</artifactId>

    <dependencies>
        <!-- Spring 相关：provided，不强制传递 -->
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-core</artifactId>
            <scope>provided</scope>
        </dependency>
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-web</artifactId>
            <scope>provided</scope>
        </dependency>

        <!-- Jackson：provided -->
        <dependency>
            <groupId>com.fasterxml.jackson.core</groupId>
            <artifactId>jackson-databind</artifactId>
            <scope>provided</scope>
        </dependency>

        <!-- Validation：provided -->
        <dependency>
            <groupId>jakarta.validation</groupId>
            <artifactId>jakarta.validation-api</artifactId>
            <scope>provided</scope>
        </dependency>

        <!-- OpenFeign：provided，只有 CommonApi 接口需要注解 -->
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-openfeign-core</artifactId>
            <scope>provided</scope>
        </dependency>

        <!-- Hutool：compile，工具类的核心依赖 -->
        <dependency>
            <groupId>cn.hutool</groupId>
            <artifactId>hutool-all</artifactId>
        </dependency>

        <!-- Lombok：compile -->
        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
        </dependency>
    </dependencies>
</project>
```

## 2. 实现 ErrorCode

ErrorCode 是一个简单的值对象，只包含错误码和消息模板：

```java
// yudao-framework/yudao-common/src/.../exception/ErrorCode.java
@Data
public class ErrorCode implements Serializable {
    private final Integer code;
    private final String msg;

    public ErrorCode(Integer code, String msg) {
        this.code = code;
        this.msg = msg;
    }
}
```

然后定义全局错误码：

```java
// yudao-framework/yudao-common/src/.../exception/enums/GlobalErrorCodeConstants.java
```

## 3. 实现 ServiceException

```java
// yudao-framework/yudao-common/src/.../exception/ServiceException.java
public final class ServiceException extends RuntimeException {
    private Integer code;
    private String message;

    public ServiceException() {}

    public ServiceException(ErrorCode errorCode) {
        this.code = errorCode.getCode();
        this.message = errorCode.getMsg();
    }

    public ServiceException(Integer code, String message) {
        this.code = code;
        this.message = message;
    }

    // fluent setter
    public ServiceException setCode(Integer code) {
        this.code = code;
        return this;
    }
}
```

`ServiceExceptionUtil` 提供工厂方法，支持 `{}` 占位符：

```java
// yudao-framework/yudao-common/src/.../exception/util/ServiceExceptionUtil.java
public class ServiceExceptionUtil {

    public static ServiceException exception(ErrorCode errorCode, Object... params) {
        String message = doFormat(errorCode.getMsg(), params);
        return new ServiceException(errorCode.getCode(), message);
    }

    // SLF4J 风格的占位符替换
    static String doFormat(String messagePattern, Object... params) {
        if (params == null || params.length == 0) {
            return messagePattern;
        }
        StringBuilder sb = new StringBuilder(messagePattern);
        for (Object param : params) {
            int index = sb.indexOf("{}");
            if (index == -1) break;
            sb.replace(index, index + 2, String.valueOf(param));
        }
        return sb.toString();
    }
}
```

## 4. 实现 CommonResult

```java
// yudao-framework/yudao-common/src/.../pojo/CommonResult.java
@Data
public class CommonResult<T> implements Serializable {
    private Integer code;
    private String msg;
    private T data;

    // ===== 工厂方法 =====

    public static <T> CommonResult<T> success(T data) {
        CommonResult<T> result = new CommonResult<>();
        result.code = GlobalErrorCodeConstants.SUCCESS.getCode();
        result.data = data;
        return result;
    }

    public static CommonResult<?> error(Integer code, String message) {
        Assert.isTrue(!GlobalErrorCodeConstants.SUCCESS.getCode().equals(code),
                "code 必须是非 0 的错误码");
        CommonResult<?> result = new CommonResult<>();
        result.code = code;
        result.msg = message;
        return result;
    }

    public static CommonResult<?> error(ErrorCode errorCode) {
        return error(errorCode.getCode(), errorCode.getMsg());
    }

    public static CommonResult<?> error(ErrorCode errorCode, Object... params) {
        return error(errorCode.getCode(),
                ServiceExceptionUtil.doFormat(errorCode.getMsg(), params));
    }

    // ===== 实例方法 =====

    @JsonIgnore
    public boolean isSuccess() {
        return GlobalErrorCodeConstants.SUCCESS.getCode().equals(code);
    }

    @JsonIgnore
    public boolean isError() {
        return !isSuccess();
    }

    public CommonResult<T> checkError() {
        if (isSuccess()) {
            return this;
        }
        throw new ServiceException(code, msg);
    }

    @JsonIgnore
    public T getCheckedData() {
        checkError();
        return data;
    }
}
```

## 5. 实现 PageParam 和 PageResult

```java
// yudao-framework/yudao-common/src/.../pojo/PageParam.java
@Data
public class PageParam implements Serializable {
    public static final Integer PAGE_SIZE_NONE = -1;

    @NotNull(message = "页码不能为空")
    @Min(value = 1, message = "页码最小值为 1")
    private Integer pageNo = 1;

    @NotNull(message = "每页条数不能为空")
    @Min(value = 1, message = "每页条数最小值为 1")
    @Max(value = 200, message = "每页条数最大值为 200")
    private Integer pageSize = 10;
}

// yudao-framework/yudao-common/src/.../pojo/PageResult.java
@Data
public final class PageResult<T> implements Serializable {
    private Long total;
    private List<T> list;

    public PageResult() {
        this.list = new ArrayList<>();
    }

    public PageResult(List<T> list, Long total) {
        this.list = list;
        this.total = total;
    }

    public static <T> PageResult<T> empty() {
        return new PageResult<>(Collections.emptyList(), 0L);
    }
}
```

## 6. 实现 BeanUtils

```java
// yudao-framework/yudao-common/src/.../util/object/BeanUtils.java
public class BeanUtils {

    public static <T> T toBean(Object source, Class<T> targetClass) {
        return cn.hutool.core.bean.BeanUtil.toBean(source, targetClass);
    }

    public static <T> T toBean(Object source, Class<T> targetClass, Consumer<T> peek) {
        T target = toBean(source, targetClass);
        if (peek != null) {
            peek.accept(target);
        }
        return target;
    }

    public static <S, T> List<T> toBean(List<S> sources, Class<T> targetClass) {
        return CollectionUtils.convertList(sources, source -> toBean(source, targetClass));
    }

    public static <S, T> PageResult<T> toBean(PageResult<S> sources, Class<T> targetClass) {
        return CollectionUtils.convertPage(sources, source -> toBean(source, targetClass));
    }

    public static void copyProperties(Object source, Object target) {
        cn.hutool.core.bean.BeanUtil.copyProperties(source, target, false);
    }
}
```

注意 `copyProperties` 的第三个参数 `false`——表示忽略 null 值，不会用 null 覆盖目标对象的已有属性。

## 7. 使用示例

### Controller 中使用 CommonResult

```java
@RestController
@RequestMapping("/system/dept")
public class DeptController {

    @Resource
    private DeptService deptService;

    @GetMapping("/get")
    public CommonResult<DeptRespVO> getDept(@RequestParam("id") Long id) {
        DeptDO dept = deptService.getDept(id);
        return CommonResult.success(BeanUtils.toBean(dept, DeptRespVO.class));
    }

    @GetMapping("/list")
    public CommonResult<PageResult<DeptRespVO>> getDeptPage(@Validated DeptPageReqVO reqVO) {
        PageResult<DeptDO> page = deptService.getDeptPage(reqVO);
        return CommonResult.success(BeanUtils.toBean(page, DeptRespVO.class));
    }
}
```

### Service 中抛出异常

```java
@Service
public class DeptServiceImpl implements DeptService {

    @Resource
    private DeptMapper deptMapper;

    @Override
    public DeptDO getDept(Long id) {
        DeptDO dept = deptMapper.selectById(id);
        if (dept == null) {
            throw exception(DEPT_NOT_FOUND);
        }
        return dept;
    }

    private void validateDeptNameUnique(Long id, Long parentId, String name) {
        DeptDO dept = deptMapper.selectByParentIdAndName(parentId, name);
        if (dept != null && !dept.getId().equals(id)) {
            throw exception(DEPT_NAME_DUPLICATE, name);
        }
    }
}
```
